Model
ğŸ”¬The model is a sequence to sequence model trained for translation.
ğŸ”¬ Strong sentence representation is obtained by passing only the overall sentence encoding to the decoder (unlike the usual translation pipeline where the decoder has access to the encoding of each token in the sentence).
ğŸ”¬ Each input sequence was jointly translated only into two languages (en, es) and not all 93 languages for two reasons -- no need for 93-way parallel corpus, no need to train 93 decoders.
ğŸ”¬ The architecture is a a biLSTM preceded by BPE encodings.
